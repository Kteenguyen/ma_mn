# Pali-Chinese Comparative Analysis Project (ma_mn)

This project contains tools and data for the comparative analysis of Middle Length Discourses (Majjhima Nikaya - MN) and their Chinese counterparts (Madhyama Agama - MA).

## Structure
- `Dashboard/`: Streamlit application for data visualization and search.
- `Phase2/`: Alignment and dictionary generation (Phase 2).
- `Phase3/`: Semantic shift, divergence analysis, and case studies (Phase 3).
- `align_v19.py`: Core alignment script.

## Getting Started
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the dashboard:
   ```bash
   streamlit run Dashboard/src/dashboard.py
   ```
